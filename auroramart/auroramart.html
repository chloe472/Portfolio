The Challenge

AuroraMart explored how a grocery e-commerce experience could feel clear, responsive, and helpful across the shopping journey, from browsing products to discovering relevant recommendations.

Key Goals

Product discovery: Help customers move quickly from browsing to relevant items.

Full-stack flow: Support core e-commerce actions through a Django-backed application.

Personalization: Integrate machine learning models for shopping recommendations.

Responsive UI: Keep the interface approachable across common screen sizes.

My Solution

I developed AuroraMart as a full-stack B2C e-commerce platform using Python and Django, pairing backend CRUD functionality with interface decisions shaped around practical grocery-shopping tasks.

The project combined UI/UX design, database-backed product flows, and recommendation logic to create a more personalized shopping experience.
